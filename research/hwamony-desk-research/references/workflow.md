# Desk Research Workflow

## Stage 1: Scope

Goal:

- convert a loose topic into a researchable question

Outputs:

- `00-project-meta.yaml`
- `01-research-brief.md`

## Stage 2: Source Strategy

Goal:

- decide what kinds of sources deserve attention first

Outputs:

- `02-source-strategy.md`

Suggested source buckets:

- official or primary sources
- reputable secondary sources
- expert commentary
- archive or background sources

## Stage 3: Batch Research

Goal:

- gather evidence in bounded batches

Outputs:

- `03-batches/batch-001.md`
- `03-batches/batch-002.md`

Each batch should record:

- focus question
- queries used
- URLs reviewed
- provisional takeaways
- open gaps

## Stage 4: Checkpoint

Goal:

- tell the user what is being researched and what has been found so far
- ask whether to continue, redirect, or stop

Do not skip this stage when the user explicitly asked for iterative feedback.

## Stage 5: Source Preservation

Goal:

- save source texts in durable markdown files

Outputs:

- `04-sources/source-index.md`
- `04-sources/raw/source-###.md`

## Stage 6: Final Synthesis

Goal:

- produce a readable research memo

Outputs:

- `05-findings.md`
- `06-bibliography.md`
- optionally `07-open-questions.md`
