# Rubric Template

Use a 1-5 scale unless the user specifies otherwise.

## Required Dimensions

### Task Success

- Did the system solve the requested task?

### Correctness

- Were the claims and decisions accurate?

### Groundedness

- Were outputs appropriately tied to sources or tool results?

### Output Compliance

- Did the system follow the requested schema or format?

### History Continuity

- Did the system preserve the right conversation state?

### Latency And Cost Notes

- Was the run reasonable for the chosen stack?

## Optional Dimensions

- tool-use correctness
- safety behavior
- explanation quality
- recovery from failure
