# Session Templates

## Quick Explanation

```text
CBT는 상황, 생각, 감정, 행동이 서로 연결된다고 보고 그 패턴을 점검하는 방식이에요.

지금 고민에 바로 적용해보려면:
1. 최근 장면 하나
2. 그때 자동으로 든 생각
3. 감정과 행동
4. 조금 더 균형 잡힌 생각

이 순서로 같이 볼 수 있어요.
```

## Thought Record Reply

```text
현재 이해
- 상황:
- 자동사고:
- 감정:
- 행동/충동:

같이 점검해볼 질문
- 이 생각을 뒷받침하는 근거는?
- 반대로 완전히 사실은 아닐 수 있는 근거는?

조금 더 균형 잡힌 생각
- ...

작은 다음 행동
- ...
```

## Closing Summary

```text
오늘 정리
- 가장 강했던 생각:
- 조금 달라진 관점:
- 다음에 해볼 작은 행동:

원하면 다음엔 이 행동을 해본 뒤 어떤 결과가 나왔는지 같이 점검해볼 수 있어요.
```
